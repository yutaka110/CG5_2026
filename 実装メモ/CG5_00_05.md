# GaussianBlur PostProcess 実装メモ

## 概要

授業資料の GaussianFilter / GaussianBlur 処理を、このプロジェクトの既存構成に合わせて
`PostProcessStack` のポストプロセスとして実装した。

資料では `CopyImage` 系の Fullscreen Shader を基に `GaussianFilter.PS.hlsl` を作り、
ガウス関数で算出した重みを使って周辺テクセルをぼかす流れだったが、
本プロジェクトにはすでに以下の仕組みがある。

- `FullscreenComposite.VS.hlsl` による fullscreen triangle 描画
- `PostProcessStack` によるポストプロセスパス管理
- `AppPostProcessPipeline` による RenderGraph へのパス登録
- `AppPipelines` による PSO / Shader 管理
- ImGui の PostProcess パネル

そのため、`CopyImage` を新規作成するのではなく、
既存のポストプロセス基盤に `GaussianBlur` パスを追加する方針にした。

また、GaussianBlur はカーネルサイズを大きくすると重くなりやすい。
たとえば 9x9 を 1 パスで処理すると 81 サンプル必要になる。
そのため、VFX エンジンに合う構成として separable blur で実装した。

今回追加したパスは以下の 2 つ。

- `GaussianBlurHorizontal.PS.hlsl`
- `GaussianBlurVertical.PS.hlsl`

9x9 相当でも、横 9 サンプル + 縦 9 サンプルの合計 18 サンプルで処理できる。
見た目は BoxBlur より自然にぼけ、負荷は 2D Gaussian より抑えられる。

## ImGui での確認方法

1. アプリを起動する。
2. VFX Engine / PostProcess パネルを開く。
3. `GaussianBlur` のチェックボックスで ON / OFF を切り替える。
   - 内部的には `GaussianBlurHorizontal` と `GaussianBlurVertical` の 2 パスをまとめて切り替える。
4. `Intensity` を操作する。
   - `0.0`: 元の画像
   - `1.0`: GaussianBlur を完全適用
5. `Kernel` でぼかし範囲を切り替える。
   - `3x3`: 半径 1 texel
   - `5x5`: 半径 2 texel
   - `7x7`: 半径 3 texel
   - `9x9`: 半径 4 texel
6. `Sigma` を操作する。
   - 値を小さくすると中心寄りの重みが強く、ぼけが控えめになる。
   - 値を大きくすると周辺にも重みが広がり、ぼけが強くなる。
