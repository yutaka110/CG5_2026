# BoxBlur PostProcess 実装メモ

## 概要

授業資料の Smoothing / Box Filter 処理を、このプロジェクトの既存構成に合わせて
`PostProcessStack` のポストプロセスとして実装した。

資料では `CopyImage` 系の Fullscreen Shader を基に `BoxFilter.PS.hlsl` を作り、
3x3 や 5x5 の周辺テクセルを平均化する流れだったが、
本プロジェクトにはすでに以下の仕組みがある。

- `FullscreenComposite.VS.hlsl` による fullscreen triangle 描画
- `PostProcessStack` によるポストプロセスパス管理
- `AppPostProcessPipeline` による RenderGraph へのパス登録
- `AppPipelines` による PSO / Shader 管理
- ImGui の PostProcess パネル

そのため、`CopyImage` を新規作成するのではなく、
既存のポストプロセス基盤に `BoxBlur` パスを追加する方針にした。

また、VFX エンジンとしては 5x5 を 1 パスで 25 サンプルするより、
横方向と縦方向に分ける separable blur の方が軽く扱いやすい。
そのため今回は、以下の 2 パス構成で実装した。

- `BoxBlurHorizontal.PS.hlsl`
- `BoxBlurVertical.PS.hlsl`

5x5 の場合でも、横 5 サンプル + 縦 5 サンプルの合計 10 サンプルで処理できる。

## ImGui での確認方法

1. アプリを起動する。
2. VFX Engine / PostProcess パネルを開く。
3. `BoxBlur` のチェックボックスで ON / OFF を切り替える。
   - 内部的には `BoxBlurHorizontal` と `BoxBlurVertical` の 2 パスをまとめて切り替える。
4. `Intensity` を操作する。
   - `0.0`: 元の画像
   - `1.0`: BoxBlur を完全適用
5. `Kernel` でぼかし範囲を切り替える。
   - `3x3`: 半径 1 texel の BoxBlur
   - `5x5`: 半径 2 texel の BoxBlur
